## IA #1: CLAUDE (Anthropic)

### Como Alan Nicolas Usa

**Frequência:** ALTA (menções regulares em últimos 30 dias)

**Use Cases:**

1. **Análise Profunda**
   - Padrão: "Explica isso para mim em 3 camadas"
   - Claude force: Reasoning estruturado, análise lógica
   - Exemplo: Entender novo framework via Claude

2. **Escrita & Estruturação**
   - Padrão: "Estrutura esse conteúdo"
   - Claude force: Qualidade de escrita, organização
   - Exemplo: Emails, posts, documentação

3. **Mentoring Content**
   - Padrão: "Como ensinar isso de forma simples?"
   - Claude force: Simplificação sem perder profundidade
   - Exemplo: Criar conteúdo educativo

### Integração com DNA Framework

**Thinking DNA:** Claude executa "3-layer analysis"
```
Claude input: "Explica como funciona aprendizado"
Claude output:
  - L1 (Superfície): O que vê na literatura
  - L2 (Padrão): Como realmente funciona
  - L3 (Princípio): Por que funciona assim
```

**Prompt pattern Alan usaria:**
```
"Claude, explica [TÓPICO] como se estivesse ensinando alguém.
Estrutura em:
1. O que a maioria pensa
2. Como realmente funciona
3. Por que funciona assim
4. Como replicar imediatamente

Sem açúcar. Verdade crua."
```

## IA #2: ChatGPT (OpenAI)

### Como Alan Nicolas Usa

**Frequência:** ALTA (alternativa rápida)

**Use Cases:**

1. **Brainstorm de Ideias**
   - Padrão: "Gera 10 ângulos para [TÓPICO]"
   - ChatGPT force: Velocidade, variedade, exploração
   - Exemplo: Ideias para vídeos, posts, conteúdo

2. **Copy/Content Rápido**
   - Padrão: "Escreve um subject line que force click"
   - ChatGPT force: Copy marketing, persuasão
   - Exemplo: Subject lines, headlines, CTAs

3. **Prototipagem Rápida**
   - Padrão: "Cria um outline para [PROJETO]"
   - ChatGPT force: Estruturação rápida, frameworks
   - Exemplo: Cursos, lançamentos, produtos

### Diferença Claude vs ChatGPT em Alan's Workflow

**Claude = Profundidade + Reasoning**
- Usa quando: análise complexa, entender raiz
- Não usa quando: precisa rapidinho

**ChatGPT = Velocidade + Exploração**
- Usa quando: gerar múltiplos ângulos
- Não usa quando: precisa qualidade garantida


## RECOMENDAÇÕES PRÁTICAS

### Para Implementar Alan's IA Usage em Seus Agentes

**1. Setup de IA Dual**
```yaml
ai_stack:
  deep_analysis: Claude
    prompt_style: "3-layer analysis"
    use_when: "need truth"

  rapid_exploration: ChatGPT
    prompt_style: "10 angles rapidinho"
    use_when: "brainstorm"

  local_testing: Ollama
    prompt_style: "experimental"
    use_when: "offline/free"
```

**2. Prompt Patterns**

**Claude Prompt (Thinking DNA):**
```
"Explica [TOPIC] em 3 camadas:
1. Superfície (O que maioria pensa)
2. Padrão (Como realmente funciona)
3. Princípio (Por que funciona)

Depois: Como replicar isso imediatamente?
Verdade crua, sem açúcar."
```

**ChatGPT Prompt (Execution DNA):**
```
"Gera 10 ângulos para [TOPIC]:
1. [Ângulo 1]
2. [Ângulo 2]
...
10. [Ângulo 10]

Para cada um: Como executar em 2 semanas?
Só o essencial, nada de fluff."
```

**3. Veto Gates (sempre após IA output)**
```javascript
vetogates = [
  { gate: "É vaga?", reject: "BULLSHIT" },
  { gate: "Tem evidência?", reject: "MAYBE" },
  { gate: "Alguém provou?", reject: "UNPROVEN" },
  { gate: "Posso replicar?", reject: "ABSTRACT" },
  { gate: "Soa verdade?", reject: "DOUBT" }
];
```
