## Technical Approach

### Task Executor Architecture
```
TaskExecutor
├─ TaskLoader (loads 205 task definitions)
├─ DependencyGraph (orders execution)
├─ TaskRunner (executes each task)
├─ CheckpointManager (saves progress)
└─ ErrorHandler (recovery + logging)
```

### Gate Enforcement
```
PreCommitHook
├─ CLI First check (no UI-only features)
├─ Agent Authority check (exclusive ops)
├─ Story-Driven check (story exists)
├─ No Invention check (spec validates)
└─ Quality First check (lint + test pass)
```

### Agent Handoff
```
HandoffArtifact {
  from_agent: string
  to_agent: string
  story_id: string
  decisions: string[]
  files_modified: string[]
  blockers: string[]
  next_action: string
}
```
