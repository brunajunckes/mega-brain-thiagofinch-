## EXPERTISE FRAMEWORK ESPERADO

### O que Alan procura em um Guest

#### 1. **Clareza do Padrão**

Guest deve conseguir responder:
```
"Qual é o padrão que você usa?"
"Se você começasse do zero, qual passo 1 seria?"
"Qual heurística te economizou mais tempo?"
"Como você valida rapidinho?"
```

**Bad answer:** "É intuição", "Fiz trial-and-error", "Virou rotina"
**Good answer:** "Sigo esse framework", "Testo em 3 semanas", "Minha regra é..."

#### 2. **Track Record Quantificado**

Guest deve ter números:
```
"Ganhei X ao fazer isto"
"Meus clientes aumentaram Y%"
"Tempo reduziu de Z para W"
"Taxa de sucesso é X%"
```

#### 3. **Replicabilidade**

Alan sempre pergunta: "Qualquer pessoa consegue fazer?"

**Se resposta é NÃO:**
- Descarta (não é padrão replicável)

**Se resposta é SIM:**
- Explora como novato faria

#### 4. **Evolução do Expertise**

Como o guest aprendeu:
```
Fase 1: Como começou?
Fase 2: Qual foi primeiro fracasso?
Fase 3: O que mudou depois?
Fase 4: Como replica agora?
```
