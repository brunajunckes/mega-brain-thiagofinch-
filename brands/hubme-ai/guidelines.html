01 // Personalidade

Os traços que definem
como nos expressamos.

→
Traço 01
Direto
Vai ao ponto. Sem introduções, sem rodeios. A primeira frase já carrega o conteúdo principal — nunca uma abertura vazia ou uma saudação que não acrescenta nada.
"Deploy concluído. 3 containers reiniciados, 0 erros."
◈
Traço 02
Inteligente
Usa terminologia técnica quando necessário. Não simplifica para parecer amigável — respeita a inteligência do interlocutor.
"RLS ativo. Zero rows expostas fora do user context."
∞
Traço 03
Parceiro
Fala como alguém da equipe, usa "nós". Não é fornecedor — é co-fundador tecnológico do que você está construindo.
"Nosso próximo passo: migrar para edge functions."
"
Como o melhor tech lead que você já teve.
◎
Traço 04
Orientado a Resultado
Foca no que foi feito e no próximo passo imediato. Não celebra processo — entrega o estado atual e o vetor de avanço.
"Story 4.2 completa. Próximo: testes de carga na edge."
▣
Traço 05
Confiável
Nunca esconde erros. Transparência é confiança. Quando algo falha, diz exatamente o que falhou, onde, e o que está sendo feito — sem minimizar, sem drama.
"Erro: timeout linha 47. Motivo: pool esgotado. Aumentando para 20."

03 // Tom por Plataforma

Mesma voz.
Calibragens diferentes.

in
LinkedIn
Profissional com personalidade. Insights reais, bastidores de construção. Sem jargão de "guru". Dados específicos, contexto real.
"Construímos um sistema que reduz custo AI em 90%. Aqui está como funciona."
𝕏
Twitter / X
Direto, provocativo, técnico. Hooks fortes, dados específicos, sem fluff. Cada tweet é uma claim que pode ser verificada.
"957 entidades. 11 agentes. 90% custo menor. É assim que uma empresa AI-first opera."
⌘
Interface UI
Ultra-direto. Estado + ação disponível. Sem artigos desnecessários. O usuário lê em movimento — cada palavra conta.
"3 tasks ativas · 0 erros · Deploy pronto →"
