**Produto:** Hubme.Ai — hub central para empresas AI-first (agentes, ferramentas, automacoes)
**Publico:** Fundadores/tech leads, startups 1-50 pessoas, $500-5.000/mes em APIs AI

**Why:** Cliente solicitou DNA estrategico completo de marca com 7 especialistas convocados.

**Decisoes chave tomadas:**

1. **One Word (Neumeier):** COMPOUND — inteligencia que se acumula como juros compostos
2. **Categoria (Ries):** AI Operating System (AI-OS) — categoria nova, sem concorrente direto
3. **Tagline recomendada:** "Where AI compounds." (EN) ou "O sistema operacional das suas AIs." (PT)
4. **Archetype:** Criador (primario) + Governante (secundario)
5. **Distinctive asset:** Cyan #22D3EE — owna a cor no espaco AI-ops
6. **Ancora de mensagem:** "$2.000/mes a mais" + "11 minutos para primeira automacao"
7. **StoryBrand:** Cliente = Sofia CTO; Vilao = fragmentacao/silo; Guia = Hubme.Ai
8. **Voz:** 80% direta / 70% acessivel / 65% casual / 95% parceiro

**How to apply:** Ao trabalhar em qualquer copy, landing page, onboarding ou comunicacao da Hubme.Ai, referenciar este file para garantir consistencia de voz, messaging e posicionamento.
